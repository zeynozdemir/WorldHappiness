# Exploratory Data Analysis of Housing Market Data

## Overview
This project performs an exploratory data analysis (EDA) on a real-world housing dataset to understand the factors associated with house prices.  
The main objective is to demonstrate data cleaning, statistical exploration, and visualization skills using Python.

---

## Data
- Source: California Housing dataset
- The dataset contains information about housing districts in California.
- Key variables include: Median house value, Median income, Housing median age, Total rooms and bedrooms, Population and households

---

## What This Project Does
- Loads and inspects a real-world dataset
- Examines data types and missing values
- Produces descriptive statistics
- Visualizes correlations between numerical variables
- Analyzes the distribution of house prices

---

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Key Insights
- Median income has a strong positive correlation with median house value
- House prices show a right-skewed distribution
- Several housing-related variables are highly correlated with each other
